# 📊 Investment Behavior Analysis – Power BI Dashboard

## Short Description
This project analyzes customer investment behavior using data cleaning in Python and interactive data visualization in Power BI.  
The dashboard helps understand investment patterns, risk preferences, and demographic trends.

## Tech Stack
- Python (Pandas)
- Google Colab
- Microsoft Power BI
- DAX

## Features and Highlights
-  Data cleaning and preprocessing using Python
-  Interactive Power BI dashboard
-  KPI cards showing key metrics
-  Slicers for filtering data
-  Multiple dashboard pages for detailed insights

## Example Insights
The dashboard provides visual insights such as:
- 💰 Most preferred investment options
- 📊 Risk tolerance by age group
- 👥 Investment distribution across demographics

## Key Questions and Insights
- What are the most popular investment choices?
- How does risk tolerance vary among customers?
- Which demographic groups invest more in specific assets?
- What trends can be observed in customer investment behavior?
